# Mongoose Wizard Project

This project is generated by Mongoose Wizard, https://mongoose.ws/wizard/
Mongoose Wizard is a visual Web dashboard builder for embedded devices.

This project can be built and run on any desktop system - Mac, Windows or Linux.
In order to build it, follow the https://mongoose.ws/docs/getting-started/build-environment/,
start command prompt, go to this directory, type "make" then run the built "firmware" binary.
